Solution Overview

This data pipeline extracts, transforms, and loads conflict data from KoboToolbox to PostgreSQL, enabling powerful analysis of war dynamics through Power BI dashboards.

How to Run

  1. Install dependencies: pip install -r requirements.txt
  2. Create .env file with credentials
  3. Execute pipeline: python pipeline.py
