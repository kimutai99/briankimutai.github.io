Project Overview

This comprehensive HR Analytics Dashboard provides executive leadership and HR professionals with actionable insights into workforce dynamics. Built with Tableau and powered by simulated HR data, the solution enables data-driven decision making across the employee lifecycle.

Key Features

  • Workforce Overview: Real-time KPIs tracking headcount, turnover rates, and hiring trends with historical comparisons
  • Diversity Analytics: Demographic breakdowns by gender, age, education level, and department with equity metrics
  • Compensation Insights: Salary distribution analysis with benchmarking against industry standards
  • Talent Mobility: Promotion patterns and career path visualization across the organization
  • Employee Directory: Searchable database with filtering capabilities for quick reference
