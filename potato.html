Potato Disease Recognition

Deep learning solution for detecting potato leaf diseases with 96% accuracy

GitHub Repository Technical Details

Solution Overview

This AI-powered application helps farmers detect potato leaf diseases early using deep learning. The system classifies potato leaves into healthy, early blight, and late blight categories with high accuracy, enabling timely intervention to protect crops.

Key Features

Image Recognition

  • Real-time classification of potato leaf images
  • Supports JPG, PNG, and JPEG formats
  • Mobile-friendly interface for field use

Deep Learning

  • Custom CNN architecture optimized for plant disease
  • Transfer learning with EfficientNet backbone
  • Data augmentation for robust performance

Disease Detection

  • Identifies Healthy leaves
  • Detects Early Blight infection
  • Recognizes Late Blight disease

Technical Implementation

Technology Stack

Python TensorFlow Flask Docker OpenCV Heroku

Model Architecture

  • Custom CNN with 5 convolutional layers
  • Batch normalization and dropout for regularization
  • GlobalAveragePooling2D for dimensionality reduction
  • Softmax output layer with 3 classes

Dataset

  • 3,000+ images from PlantVillage dataset
  • Balanced classes (1,000 images per category)
  • 80/10/10 train/validation/test split
  • Augmented with rotation, zoom, and flip transforms

Deployment

  • Flask backend for model serving
  • Docker container for portability
  • Heroku cloud deployment
  • CI/CD pipeline with GitHub Actions

Installation & Setup

# Clone the repository
git clone https://github.com/kimutai99/Potato-Leaf-Classification.git
cd Potato-Leaf-Classification

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py

Docker Deployment

# Build the Docker image
docker build -t potato-disease-app .

# Run the container
docker run -p 5000:5000 potato-disease-app

# Access the application
http://localhost:5000
